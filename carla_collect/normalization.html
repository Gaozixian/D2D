驾驶数据归一化方法详解

一、Min-Max 归一化（最小-最大归一化）

将特征值从原始范围映射到指定目标区间（默认 [0,1]），保留特征的相对比例，适用于有明确物理范围的特征（如油门、刹车）。

1. 核心计算公式

\[ \text{归一化后的值} = \frac{\text{原始值} - \text{原始最小值}}{\text{原始最大值} - \text{原始最小值}} \times (\text{target_max} - \text{target_min}) + \text{target_min} \]

参数说明：

  • 原始最小值/最大值：该特征在训练集中的最小/最大值
  • target_min/target_max：目标区间（默认 0/1）

2. 实际数据计算实例（转向角 steer）

已知：

  • 原始最小值 = -0.731285
  • 原始最大值 = 0.632950
  • 目标区间 = [0,1]
  • 某条数据原始转向角 = 0.0

计算：

\[ \text{steer_minmax} = \frac{0.0 - (-0.731285)}{0.632950 - (-0.731285)} \times (1 - 0) + 0 \approx 0.53604 \]

结果：该条数据的转向角 Min-Max 归一化值为 0.53604（在 [0,1] 区间内）

3. 特殊情况处理

若某列所有值相同（如刹车 brake 列全为 0），则 原始最大值 - 原始最小值 = 0，为避免除以 0 错误，直接将归一化值设为 target_min（默认 0）。

4. 代码实现示例

def min_max_normalize(df, col, feature_range=(0, 1)): original_min = df[col].min() original_max = df[col].max() target_min, target_max = feature_range if original_max - original_min < 1e-10: return [target_min] * len(df) normalized = (df[col] - original_min) / (original_max - original_min) * (target_max - target_min) + target_min return normalized

二、Z-Score 标准化（标准差标准化）

将特征值转换为符合标准正态分布（均值=0，标准差=1）的数值，削弱异常值影响，适用于无明确范围的特征（如转向角、速度）。

1. 核心计算公式

\[ \text{标准化后的值} = \frac{\text{原始值} - \text{原始均值}}{\text{原始标准差}} \]

参数说明：

  • 原始均值：该特征在训练集中的平均值
  • 原始标准差：该特征在训练集中的标准差

2. 实际数据计算实例（速度 speed_kmh）

已知：

  • 原始均值 ≈ 28.79688
  • 原始标准差 ≈ 5.879584
  • 某条数据原始速度 = 0.000009 km/h

计算：

\[ \text{speed_kmh_zscore} = \frac{0.000009 - 28.79688}{5.879584} \approx -4.89777 \]

结果：该条数据的速度 Z-Score 标准化值为 -4.89777（均值为 0，标准差为 1）

3. 特殊情况处理

若某列所有值相同（标准差=0），直接将标准化值设为 0（符合标准正态分布的均值）。

4. 代码实现示例

def zscore_normalize(df, col): original_mean = df[col].mean() original_std = df[col].std() if original_std < 1e-10: return [0.0] * len(df) normalized = (df[col] - original_mean) / original_std return normalized

三、两种方法对比

对比维度 Min-Max 归一化 Z-Score 标准化
结果范围 可自定义（默认 [0,1]） 无固定范围（通常 [-3,3]）
异常值影响 大（压缩正常数据范围） 小（均值/标准差缓冲）
适用特征 有明确物理范围（油门、刹车） 无明确范围（转向角、速度）
适用模型 神经网络、CNN/RNN（对范围敏感） 线性回归、SVM、KNN（对分布敏感）
逆归一化难度 简单（需保存最值） 简单（需保存均值/标准差）

四、使用建议

1. 训练集/验证集/测试集必须使用训练集的归一化参数，不可单独计算测试集参数；

2. 神经网络优先选 Min-Max 归一化，传统机器学习模型优先选 Z-Score 标准化；

3. 模型输出需通过逆归一化还原为物理值（如 km/h、转向角度）。
